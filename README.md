# EV Charging Dashboard

A React + TypeScript application that simulates and visualizes EV charging data. Users can configure simulation parameters, view energy consumption, charging events, and power demand over different time intervals.

The project is built with **React**, **TypeScript**, **Tailwind CSS**, and **Recharts** for data visualization.

---

## Installation

1. Clone the repository

2. Open the directory

```bash
cd ev-charging-dashboard
```

3. Install dependencies

```bash
npm install
```

4. Run

```bash
npm run dev
```

5. Open the app in you browser (The default port for Vite is 5173)
